# 変更履歴 / Changelog

## v12.0.0 (2026-06)

### 配色・デザイン
- ページ全体の配色を白黒ベースに変更
- 3Dモデルの色を黒に近いグレー (#3a3a3a) に変更
- ボタンの色を黒 (#111827) に変更
- 形状選択（ホールパターン・断面形状）のハイライト色を青から黒に変更
- ステータス表示色を新しい配色に合わせて調整

### コード品質改善（GitHub Copilotレビュー反映・1回目）
- 入力値バリデーションを追加（`getParams()`）
  - 直径・枠の太さ・形状分割レベルの範囲チェック
- try/catchによるエラーハンドリングを追加（`safeUpdate()`）
  - エラー発生時にステータス表示でメッセージを通知
- STLダウンロード処理をBlobベースに変更（base64データURLから移行）
- 変数名 `sub` を `subd` にリネーム（グローバル関数とのシャドーイング解消）
- 未使用の `b64` 関数を削除
- ステータス表示に `aria-live="polite"` を追加（アクセシビリティ向上）

### コード品質改善（GitHub Copilotレビュー反映・2回目）
- `preserveDrawingBuffer` を `false` に変更（描画の軽量化）
- `keyLight.target` の追加処理を初期化時の一度だけに整理（再描画ごとの重複追加を解消）
- タッチ操作対応を追加
  - スマートフォン・タブレットで1本指ドラッグによる回転操作
  - 2本指ピンチ操作によるズーム

### ドキュメント
- `README.md` を追加
  - GitHub Pages の公開先リンクを記載
  - 主な機能、パラメータ、操作方法、STLエクスポート手順を整理
  - デスクトップ操作とタッチ操作の説明を追加
  - 今後の改善候補を Development セクションに記載
